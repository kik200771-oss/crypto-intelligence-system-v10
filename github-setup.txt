GitHub Setup Commands for CIS V10.0-r1
=========================================

After creating repository on GitHub, run:

# Add remote origin (replace YOUR_USERNAME with actual GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/crypto-intelligence-system-v10.git

# Push to GitHub
git branch -M main
git push -u origin main

# Verify remote connection
git remote -v
