# 🖥️ Руководство по интерфейсам системы

В вашей системе анализа криптовалют доступны несколько способов просмотра результатов анализа и торговых сигналов.

## 📊 1. Jupyter Notebook (Рекомендуется)

**Самый удобный и интерактивный способ анализа**

### Запуск:
```bash
jupyter notebook notebooks/crypto_analysis_demo.ipynb
```

### Возможности:
- 📈 **Интерактивные графики** - цены, объемы, индикаторы
- 🔍 **Детальный анализ** - пошаговое выполнение
- 📊 **Визуализация данных** - красивые графики Plotly
- 🤖 **Обучение моделей** - с визуализацией результатов
- 📝 **Документация** - объяснения каждого шага

### Что вы увидите:
- Свечные графики с техническими индикаторами
- RSI, MACD, Bollinger Bands в виде графиков
- Результаты машинного обучения с метриками
- Корреляционные матрицы
- Прогнозы с уровнями уверенности

## 🖥️ 2. Терминальный интерфейс

**Красивый вывод в консоли с цветовой индикацией**

### Разовый анализ:
```bash
python main.py --terminal --symbol BTCUSDT
```

### Мониторинг в реальном времени:
```bash
python main.py --monitor --symbol BTCUSDT
```

## 🌐 3. Веб-дашборд (опционально)

**Интерактивный веб-интерфейс для мониторинга**

### Установка зависимостей:
```bash
pip install dash plotly dash-bootstrap-components
```

### Запуск:
```bash
python main.py --dashboard
```

Откройте браузер: http://127.0.0.1:8050

### Возможности:
- Выбор криптовалюты из списка
- Настройка интервала времени
- Автообновление каждые 5 минут
- Торговые сигналы в реальном времени
- ML прогнозы с вероятностями
- Интерактивные графики

## 📱 4. Telegram бот (для уведомлений)

**Получение сигналов в Telegram**

### Настройка:
1. Создайте бота через @BotFather
2. Получите токен
3. Измените в `src/interfaces/telegram_bot.py`:
   ```python
   BOT_TOKEN = "ваш_токен_здесь"
   ```

### Команды бота:
- `/start` - приветствие и меню
- `/price BTC` - текущая цена Bitcoin
- `/analyze BTC` - полный анализ с сигналами
- `/signals` - быстрые сигналы по топ монетам
- `/portfolio` - обзор рынка
- `/subscribe` - подписка на уведомления

## 🎯 Какой интерфейс выбрать?

### Для обучения и изучения:
**→ Jupyter Notebook** - лучший выбор для понимания процесса

### Для быстрой проверки:
**→ Терминальный интерфейс** - быстро и информативно

### Для постоянного мониторинга:
**→ Веб-дашборд** - удобный для наблюдения в браузере

### Для уведомлений на мобильный:
**→ Telegram бот** - получайте важные сигналы где бы вы ни были

## 🔧 Примеры использования

### Быстрый анализ Bitcoin:
```bash
python main.py --terminal --symbol BTCUSDT
```

### Мониторинг нескольких монет:
```bash
python main.py --monitor  # BTC + ETH по умолчанию
```

### Веб-дашборд с автообновлением:
```bash
python main.py --dashboard
```

### Комбинированный подход:
1. Сначала Jupyter Notebook для изучения
2. Затем терминал для быстрых проверок
3. Веб-дашборд для постоянного мониторинга
4. Telegram бот для мобильных уведомлений
